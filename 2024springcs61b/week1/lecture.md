**Static vs. Non-Static Methods**

**Static Methods**

```java
public class Dog {
    public static void makeNoise() {
        System.out.println("Bark!");
    }
}
public class DogLauncher {
    public static void main(String[] args) {
        Dog.makeNoise();
    }
}
```
